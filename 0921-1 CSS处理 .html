CSS基本属性处理

            1.css() 获取css属性值。
            $(selector).css(name); 获取匹配选择器的元素指定css属性的值。
            
            可以一次性设置多个属性值。

            $(selector).css({"properties":"value1","properties":"value2"});


            2.CSS尺寸的属性处理

                1) height() 和 width() 获取或设置css属性 height() width() 单位是像素。
                设置和获取 content 的 widht 和 height
                说明：需要注意的是，height() 和 width() 方法获取的都是元素在页面中的实际高度值和宽度值，与设置的不一定相同。
                $("div").width();  获取第一个div的width; 获取值无单位。
                $("div").width(30);  设置所有的div的width为30px;
                $("div").width("70%");  设置所有div的width为父元素的70%;

                2) innerHegith()  innerWidth()
                获得或设置元素的实际高度和宽度值(content+padding))。不包含border和margin
                一般只用来获取。
                设置宽度和高度时，修改content的宽高;

                3) outerHeight() outerWidth()
                获取或设置元素的实际高度和宽度值。
                一般只用来获取。
                参数true 获取宽高包含 margin
                $("div").outerHeight(true)

            3.css位置属性处理  默认单位px

                1) offset() 获取或设置当前元素的相对窗口偏移量
                $("p").offset().top; // 获得第一个p距离窗口坐上的偏移距离。
                $("p").offset().left; // 获得第一个p距离窗口左边的偏移距离。
                $(selector).offset(); [object object] 需要top left属性来获取值。
                $(selector).offset({"top":100,"left":100});  设置偏移量。

                2) position() 获取当前元素相对最近的定位元素的偏移量
                $("p").pisition().top; // 获得第一个p距离父元素上边的偏移距离。
                $("p").posiiton().left; // 获得第一个p距父元素左边的偏移距离。

                3) scrollTop()  scrollLeft()  获取或设置滚动条的位置
                $("div:eq(1)").scrollTop(); // 获得滚动条的垂直距离
                $("div:eq(1)").scrollLeft(); // 获得滚动条的水平距离
